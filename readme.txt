== Description ==
Simple Team Widgets is an elementor add-on to showcase your Team Widget style/design. This is an simple and flexible way to add new elements/widgets to Elementor Page Builder.

Note: This plugin is a widget of Elementor Page Builder and will only work with Elementor Page Builder installed.
